🚀 실제 배포 방법

이 HTML 파일 하나만 있으면 누구나 접속 가능한 공개 페이지로 쉽게 배포할 수 있습니다.
GitHub Pages
무료이며 학과 홈페이지처럼 공유하기 좋습니다. 이 파일 이름을 index.html로 저장해 올리면 됩니다.
Vercel
드래그 앤 드롭만으로 빠르게 배포할 수 있어 비전공자에게도 쉽습니다.
LMS 업로드
학교 LMS나 강의 게시판에 링크를 올리면 학생들이 로그인 없이 외부 브라우저에서 바로 사용할 수 있습니다.

💡 활용 아이디어

교수자와 학생 모두에게 유용하도록 설계했습니다.
학생은 발표 주제 찾기 전에 먼저 이 추천기를 사용해 관심 있는 주제를 좁힐 수 있습니다.
조별 과제에서는 여러 명이 각자 추천 결과를 저장한 뒤, 가장 좋은 아이디어를 골라 팀 주제로 삼을 수 있습니다.
교수자는 수업 시작 5분 전에 즉석 활동을 뽑아 아이스브레이킹이나 토론 과제로 쓸 수 있습니다.
이 페이지는 단일 HTML 파일로 구성되어 배포가 쉽고, 학생이 직접 저장·복사·공유할 수 있도록 설계되었습니다.

배포 안내

1. 이 파일을 index.html로 저장합니다.
2. GitHub Pages 또는 Vercel에 업로드합니다.
3. 생성된 주소를 학생들에게 공유하면, 로그인 없이 바로 접속 가능합니다.
4. 학생이 저장한 내용은 서버가 아니라 각자 브라우저에만 저장됩니다.
